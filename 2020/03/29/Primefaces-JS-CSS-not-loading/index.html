Primefaces JS/CSS not loading

2020-03-29 | tech issue

PrimeFaces will auto-include the necessary JS/CSS resources in <h:head>. However, if you don’t have that tag and use a “plain HTML” <head>, the included JS/CSS will not be loaded.

This mistake should also have been logged in the server log as below:
One or more resources has the target of ‘head’ but not ‘head’ component has been defined within the view

Solution:

Put JS/CSS resources in <h:head> as follows.
