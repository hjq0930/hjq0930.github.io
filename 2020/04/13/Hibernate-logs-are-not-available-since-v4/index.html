Hibernate logs are not available since v4

2020-04-13 | tech issue

Since v4, Hibernate uses jboss-logging as logging framework. This framework delegates to one of the following: log4j2, log4j, slf4j, Java Utils Logging.

Diagnostics:

My application uses SLF4J/Log4j as logging library but jboss-logging can’t seem to find this library. The reason is probably that jboss-logging is loaded from the Weblogic server modules classloader whereas log4j is loaded from the application classloader.
